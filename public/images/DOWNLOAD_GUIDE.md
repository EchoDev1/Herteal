## Download Instructions

### From Unsplash:
1. Click on an image you like
2. Click the green "Download free" button
3. The image will download in high resolution
4. Rename it to the appropriate name (e.g., editorial-1.jpg)
5. Move it to: `C:\Users\USER\Desktop\herteal\public\images\`

### From Pexels:
1. Click on an image you like
2. Click the green "Free Download" button
3. Select "Original" size
4. The image will download
5. Rename it to the appropriate name (e.g., editorial-2.jpg)
6. Move it to: `C:\Users\USER\Desktop\herteal\public\images\`

## Image Optimization Tips

After downloading, consider optimizing your images for web performance:

1. **Resize if needed:**
   - Editorial-1: Max width 2000px
   - Editorial-2: Max width 2000px
   - Editorial-3, 4, 5: Max width 1200px

2. **Compress images:**
   - Use tools like TinyPNG.com or Squoosh.app
   - Target file size: 200-500KB per image
   - Maintain quality at 85-90%

3. **Convert to WebP (optional but recommended):**
   - Better compression and quality
   - Next.js will automatically handle this with the Image component
